# RLNT WEAPON HUD

### INFISTAR INSTALLATION

- Open your infiSTAR *AHconfig.sqf* and add the following dialog number to your *ALLOWED_Dialogs*:
	```
	7211
	```
- You're done and you can go back to the [main post][mainpost]!
